# Вариант гроукомпакта: компактное решение

![Компактный гров](/images/guides/grow-compact.jpg)

## Концепция гроукомпакта

Гроукомпакт — это миниатюрный гроубокс, спроектированный для ограниченного пространства. Типичные размеры: 30–50 см в ширину и глубину, 50–80 см в высоту. Несмотря на скромные габариты, при грамотном подходе в таком боксе можно вырастить полноценный урожай.

Гроукомпакт подходит, если:

- Вы живёте в маленькой квартире и не можете выделить место под полноразмерный бокс
- Хотите попробовать гров с минимальными вложениями
- Выращиваете 1 компактное растение
- Нужен «стелс» — незаметный бокс, замаскированный под бытовой предмет

## Подходящие размеры

| Размер (Ш×Г×В) | Объём | Подходит для | Комментарий |
|-----------------|-------|-------------|-------------|
| 30×30×50 см | 45 л | Микрозелень, рассада | Только вегетация |
| 35×35×60 см | 74 л | 1 миниатюрное растение | Требует интенсивной тренировки |
| 40×40×70 см | 112 л | 1 компактное растение | Оптимальный минимум для полного цикла |
| 50×50×80 см | 200 л | 1–2 растения | Комфортный гроукомпакт |

> **Важно:** в компактном пространстве каждый сантиметр на счету. Высота лампы + расстояние от лампы до верхушки + высота горшка = «мёртвая зона». В боксе высотой 70 см для растения остаётся всего 30–35 см.

## Материалы для корпуса

Для компактного бокса подходят:

- **Пластиковый контейнер** — системный блок ПК, пластиковый ящик, корпус от тумбочки
- **Фанера 8–10 мм** — легче ЛДСП, достаточно прочная для малых размеров
- **Металлический корпус** — системный блок компьютера, серверный шкаф, сейф
- **Готовая мебель** — прикроватная тумба, мини-шкафчик, кухонная тумба

Внутренние стенки оклеиваются пенофолом или окрашиваются белой матовой краской.

## Вентиляция для малых объёмов

В компактном пространстве перегрев — главная проблема. Объём воздуха мал, и даже маломощная лампа быстро его нагревает.

### Схема вентиляции

```
[Приток] ──→ [Бокс] ──→ [Вытяжной вентилятор] ──→ [Угольный фильтр] ──→ [Наружу]
 (низ)                         (верх)
```

### Выбор вентилятора

Для бокса до 200 литров достаточно:

- **Компьютерные вентиляторы 120 мм** — тихие, дешёвые, 12В питание
- **Канальный вентилятор 75–100 мм** — для установки угольного фильтра
- **USB-вентиляторы** — для самых компактных боксов

Расчёт: объём бокса должен обновляться **каждые 1–3 минуты**. Для бокса 100 литров нужен вентилятор с производительностью от 2 м³/час (с учётом сопротивления фильтра).

### Угольный фильтр

Для малых объёмов промышленные фильтры слишком велики. Варианты:

- **Самодельный фильтр** из сетчатого цилиндра, заполненного активированным углём
- **Компактный фильтр 75 мм** — существуют модели длиной 15–20 см
- **ONA-гель** — поглотитель запахов как дополнение (не замена) фильтру

## Выбор освещения

### LED-ленты

Бюджетный вариант для самых компактных боксов. Используйте ленты полного спектра или комбинацию тёплого белого (3000K) и холодного белого (6500K).

- **Мощность:** 15–30 Вт для бокса 30×30 см
- **Плюсы:** минимальная высота профиля (1–2 см), равномерное распределение, низкий нагрев
- **Минусы:** невысокая эффективность по сравнению с квантум-бордами

### Квантум-борды малой мощности

Оптимальный выбор для гроукомпакта. Компактные платы с высокоэффективными диодами Samsung LM301B или аналогами.

- **Мощность:** 30–60 Вт для бокса 40×40 см
- **Плюсы:** высокая эффективность (2.5–3.0 мкмоль/Дж), полный спектр
- **Минусы:** выше стоимость, выделяют больше тепла, чем ленты

### Рекомендуемая мощность по размеру бокса

| Размер бокса | LED-лента | Квантум-борд |
|-------------|-----------|-------------|
| 30×30 см | 15–25 Вт | 25–35 Вт |
| 40×40 см | 25–40 Вт | 40–60 Вт |
| 50×50 см | 40–60 Вт | 60–80 Вт |

## LST и тренировки для ограниченного пространства

В компактном боксе контроль высоты — это необходимость, а не опция. Без тренировок растение упрётся в лампу.

### Low Stress Training (LST)

Основная техника для гроукомпакта. Суть: пригибание стеблей в горизонтальное положение с помощью проволоки или креплений.

1. Начинайте LST с 3–4 узла (когда растение достигнет 10–15 см)
2. Аккуратно пригните верхушку к краю горшка и зафиксируйте
3. По мере роста боковых побегов — пригибайте и их
4. Цель — плоский ровный «ковёр» из верхушек на одной высоте

### Топпинг / FIM

Удаление верхушки для стимуляции роста боковых побегов:

- **Топпинг** — срезание верхней точки роста. Из одной верхушки вырастают две.
- **FIM** — удаление 75% верхушки. Может дать 3–4 новых побега.

Проводите на стадии 4–5 узлов. Сочетайте с LST для максимального контроля высоты.

### ScrOG (Screen of Green)

Установка сетки на высоте 15–20 см от горшка. Побеги направляются горизонтально под сеткой, формируя равномерный полог. Идеальная техника для компактного пространства.

## Примеры реализации

### Системный блок ПК

Корпус системного блока (стандартный ATX) имеет размеры примерно 45×20×45 см. Подходит для одного миниатюрного растения с интенсивной тренировкой.

- LED-лента 20 Вт на крышке
- Штатные вентиляторы 120 мм для охлаждения
- Пенофол на внутренних стенках

### Кухонная тумба

Тумба 40×40×70 см — один из лучших вариантов стелс-бокса. Внешне выглядит как обычная мебель.

- Квантум-борд 50 Вт на потолке
- Канальный вентилятор 100 мм в задней стенке
- Самодельный угольный фильтр
- Дверца с уплотнителем для светоизоляции

### Пластиковый контейнер

Непрозрачный контейнер 50×35×40 см для хранения вещей. Самый бюджетный вариант.

- LED-лента по периметру крышки
- Компьютерные кулеры 80 мм на боковой стенке
- Стенки оклеены фольгоизолом

## Советы для микрогрова

1. **Выбирайте компактные генетики** — автоцветы, карликовые сорта, индика-доминантные гибриды
2. **Используйте маленькие горшки** — 3–5 литров достаточно для ограничения размера растения
3. **Начинайте тренировку рано** — не давайте растению вытянуться
4. **Контролируйте температуру** — в малом объёме перегрев наступает быстро
5. **Поливайте чаще, но меньше** — в маленьких горшках субстрат высыхает быстрее
6. **Используйте таймер** — автоматизируйте хотя бы свет, чтобы снизить внимание к боксу
