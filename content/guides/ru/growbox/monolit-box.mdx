# Монолит: строим капитальный гроубокс

![Капитальный гроубокс](/images/guides/monolit-box.jpg)

## Концепция «Монолита»

«Монолит» — это капитальный гроубокс, построенный из строительных материалов на долгие годы эксплуатации. В отличие от тентов, он обеспечивает:

- Полную звукоизоляцию
- Отличную теплоизоляцию
- Жёсткую конструкцию для тяжёлого оборудования
- Возможность интеграции в интерьер
- Максимальную светоизоляцию

Проект рассчитан на бокс размером **80×80×180 см** — оптимальный размер для 2–3 растений.

## Планирование и проект

Перед началом строительства определите:

1. **Место установки** — ровный пол, доступ к розетке, возможность вывести вентиляцию
2. **Размеры** — исходите из доступного пространства и планируемого количества растений
3. **Бюджет** — капитальный бокс дороже тента, но служит значительно дольше
4. **Доступ** — дверь должна обеспечивать полный доступ к внутреннему пространству

Нарисуйте чертёж с указанием всех размеров, расположения вентиляционных отверстий, электрики и двери.

## Необходимые инструменты

- Шуруповёрт / дрель
- Свёрла по дереву (5 мм, 8 мм)
- Коронка по дереву 100–110 мм (для вентиляционных отверстий)
- Электролобзик (если делаете раскрой самостоятельно)
- Рулетка, угольник, карандаш
- Строительный нож
- Уровень

## Пошаговая сборка корпуса

### Шаг 1: Раскрой деталей

Для бокса 80×80×180 см потребуются:

- **Боковые стенки** — 2 шт, 80×180 см
- **Верх и низ** — 2 шт, 80×76.8 см (минус толщина двух боковых стенок)
- **Задняя стенка** — 1 шт, 76.8×178.4 см
- **Дверь** — 1 шт, 76.8×178.4 см (или две створки по 76.8×89.2 см)

### Шаг 2: Сборка каркаса

1. Соедините дно и боковые стенки с помощью конфирматов, предварительно засверлив отверстия
2. Усильте соединения мебельными уголками изнутри
3. Установите верхнюю панель аналогичным образом
4. Закрепите заднюю стенку — она придаёт жёсткость всей конструкции
5. Проверьте прямоугольность с помощью уровня и замера диагоналей

### Шаг 3: Установка двери

1. Разметьте положение петель — сверху, снизу и по центру
2. Прикрутите петли к двери, затем к боковой стенке
3. Установите магнитные защёлки для плотного прилегания
4. По периметру двери наклейте уплотнитель (D-профиль) для светоизоляции

## Вентиляционные отверстия

Два основных отверстия диаметром 100 мм:

- **Приточное** — в нижней части задней или боковой стенки (10–15 см от пола)
- **Вытяжное** — в верхней части, противоположная сторона (10–15 см от потолка)

Вырезайте коронкой по дереву. Установите фланцы и закрепите саморезами. Промажьте стыки герметиком.

Для пассивного притока можно сделать 2–3 отверстия диаметром 50 мм вместо одного на 100 мм.

## Электрика

### Схема электропроводки

Внутри бокса должно быть:

- **Распределительная коробка** с автоматом на 10А
- **2–3 розетки** — для лампы, вентилятора, дополнительного оборудования
- **Ввод кабеля** — через герметичное отверстие в стенке

Правила безопасности:

- Используйте кабель ВВГнг-Ls сечением не менее 1.5 мм²
- Все соединения — через клеммники Wago, никаких скруток
- Заземление обязательно
- Установите УЗО на линию питания бокса
- Все розетки — с заземляющим контактом, желательно влагозащищённые (IP44)

## Внутренняя отделка

1. **Герметизация швов** — промажьте все стыки силиконовым герметиком
2. **Утепление** (опционально) — оклейте стенки пенофолом 5 мм фольгой внутрь
3. **Отражающее покрытие** — если не используете пенофол, покройте стенки белой матовой краской в 2–3 слоя или наклейте майлар
4. **Гидроизоляция пола** — уложите на дно поддон или застелите плотной плёнкой с бортиками

## Установка оборудования

### Свет

Закрепите подвесы или крюки на верхней панели. Используйте регулируемые тросики для изменения высоты лампы. Для бокса 80×80 см оптимальная мощность LED — 150–250 Вт.

### Вентиляция

1. Установите канальный вентилятор на вытяжку
2. Подключите угольный фильтр перед вентилятором
3. Соедините гофрой: фильтр → вентилятор → вывод наружу
4. Приточное отверстие оставьте пассивным (разрежение затянет воздух)

### Дополнительное

- Термогигрометр с выносным датчиком
- Вентилятор обдува на клипсе
- Таймер для света
- Регулятор оборотов для канального вентилятора

## Финальная проверка

Перед первым запуском убедитесь:

- [ ] Дверь закрывается плотно, свет не пробивается наружу
- [ ] Вентиляция работает, воздух обновляется
- [ ] Электрика исправна, УЗО срабатывает при тесте
- [ ] Внутри нет острых краёв и выступающих саморезов
- [ ] Поддон не протекает
- [ ] Все стыки загерметизированы
