# Диагностика проблем растений: общий подход

![Диагностика](/images/guides/plant-diagnosis.jpg)

Видишь проблему на листьях, но не знаешь что это? Систематический подход поможет диагностировать за 5 минут. Используй этот чек-лист каждый раз.

## Система диагностики за 5 шагов

### Шаг 1: Где симптомы?

**Вопрос:** На каких листьях появилась проблема?

**Нижние старые листья:**
→ Мобильные элементы (N, P, K, Mg, Zn)
→ Растение «высасывает» элемент из старых листьев

**Верхние новые листья:**
→ Немобильные элементы (Ca, Fe, S, B, Mn)
→ Или экологический стресс (свет, температура)

**Все листья:**
→ Перекорм (nutrient burn)
→ pH проблемы (nutrient lockout)
→ Вредители
→ Экологический стресс

**Таблица быстрой диагностики:**

| Где | Что | Вероятная проблема |
|-----|-----|-------------------|
| Низ | Желтеют | Азот (N) или недолив |
| Низ | Коричневые края | Калий (K) |
| Низ | Межжилковый хлороз | Магний (Mg) |
| Верх | Желтые с зелёными жилками | Железо (Fe) |
| Верх | Коричневые пятна | Кальций (Ca) |
| Все | Обожжённые кончики | Перекорм / pH |
| Все | Точки, паутина | Вредители |

### Шаг 2: Какой цвет/форма?

**Жёлтые листья:**
- Низ → Азот, недолив
- Верх → Железо, Сера
- Межжилковый (низ) → Магний

**Коричневые:**
- Края (низ) → Калий
- Пятна (верх) → Кальций
- Кончики (все) → Перекорм

**Скрученные:**
- Вниз («когти») → Избыток азота, стресс от ветра
- Вверх («каноэ») → Тепловой стресс
- Внутрь → Недолив, низкая влажность

**Вялые:**
- Тяжёлые, зелёные → Перелив
- Сухие, хрустящие → Недолив

### Шаг 3: Проверь условия

**ОБЯЗАТЕЛЬНО проверь ПЕРВЫМ:**

✅ **pH раствора:**
   - Грунт: 6.0-6.5
   - Гидра/кокос: 5.8-6.2
   - **80% проблем из-за pH!**

✅ **EC (если есть EC-метр):**
   - Рассада: 0.4-0.8
   - Вегетация: 0.8-1.6
   - Цветение: 1.4-2.0
   - >2.2 = перекорм

✅ **Температура:**
   - 20-28°C (68-82°F)
   - <18°C или >30°C = стресс

✅ **Влажность:**
   - Вегетация: 50-70%
   - Цветение: 40-50%
   - >70% = риск плесени

✅ **Полив:**
   - Грунт: верхний слой сухой?
   - Вес горшка: лёгкий или тяжёлый?

### Шаг 4: Когда началось?

**Внезапно (1-2 дня):**
- Стресс (свет, температура, ветер)
- Перелив / недолив
- Механическое повреждение

**Медленно (неделя+):**
- Дефицит питательных веществ
- pH проблемы
- Root bound (маленький горшок)

**Прогрессирует быстро:**
- Вредители (клещ удваивается каждые 3 дня)
- Болезни (плесень, гниль)
- Критический дефицит

### Шаг 5: Есть ли другие признаки?

**Паутина, точки, движущиеся насекомые:**
→ Вредители

**Белый налёт, плесень:**
→ Болезни

**Гнилостный запах:**
→ Корневая гниль

**Растение не реагирует на подкормки:**
→ pH проблемы (nutrient lockout)

## Чек-лист систематической диагностики

Распечатай и держи в гроуруме:

### 1. Условия среды ✅

- [ ] pH: _____ (норма: 6.0-6.5 грунт, 5.8-6.2 гидра)
- [ ] EC: _____ (норма: 0.8-2.0)
- [ ] Температура: _____°C (норма: 20-28°C)
- [ ] Влажность: _____% (норма: 40-70%)
- [ ] Свет: расстояние _____ см, мощность _____ W

### 2. Полив ✅

- [ ] Последний полив: _____ дней назад
- [ ] Грунт: сухой / влажный / мокрый
- [ ] Вес горшка: лёгкий / нормальный / тяжёлый
- [ ] Дренаж: есть / нет

### 3. Симптомы ✅

- [ ] Где: низ / верх / все листья
- [ ] Цвет: жёлтый / коричневый / пятна
- [ ] Форма: скручены / вялые / нормальные
- [ ] Когда началось: _____ дней назад

### 4. Кормление ✅

- [ ] Последняя подкормка: _____ дней назад
- [ ] Доза: 1/4 / 1/2 / полная / больше
- [ ] Тип удобрений: Grow / Bloom / другое

### 5. Вредители/Болезни ✅

- [ ] Осмотрел с лупой: да / нет
- [ ] Нижнюю сторону листьев: чистая / есть признаки
- [ ] Запах: нормальный / затхлый / гнилостный

## Частые ошибки диагностики

### Ошибка 1: Диагноз по одному симптому

**Неправильно:**
«Листья жёлтые = азот!» → Дал азот → Стало хуже

**Правильно:**
Проверил pH → был 7.5 → Азот не усваивался из-за pH → Скорректировал pH → Проблема решилась

### Ошибка 2: Не проверил условия

**Неправильно:**
Сразу ищешь дефицит → Покупаешь удобрения

**Правильно:**
Проверил pH, EC, температуру → Нашёл причину → Исправил условия

### Ошибка 3: «Лечение» подкормками

**Неправильно:**
Растение плохое → Дам больше удобрений → Стало хуже (перекорм)

**Правильно:**
Растение плохое → Проверил pH/EC → Промыл субстрат → Начал с малых доз

### Ошибка 4: Паника и радикальные действия

**Неправильно:**
Увидел проблему → Пересадил, промыл, поменял удобрения за день → Растение в шоке

**Правильно:**
Увидел проблему → Диагностировал → Исправил ОДНУ вещь → Жду 3-5 дней → Оцениваю

## Таблица приоритетов действий

| Проблема | Срочность | Действие |
|----------|-----------|----------|
| Вредители (клещ, трипсы) | Критично! | Немедленно изолируй + обработка |
| Плесень (серая гниль) | Критично! | Срежь поражённое + снизь RH |
| Перелив + корневая гниль | Очень срочно | Прекрати полив + H₂O₂ |
| Тепловой стресс (>32°C) | Срочно | Снизь температуру сейчас |
| Световой ожог | Срочно | Подними лампу |
| Дефицит азота | Средне | Подкорми (5-7 дней восстановление) |
| Дефицит других элементов | Средне | Проверь pH → подкорми |
| Лёгкий перекорм | Низко | Пропусти полив с удобрениями |

## Золотое правило диагностики

**Сначала проверь простое:**
1. pH — 80% проблем!
2. Полив — перелив/недолив
3. Температура и влажность
4. Только потом удобрения

**Принцип Оккама:**
«Самое простое объяснение обычно правильное»
- Листья вялые? → Полив, не дефицит магния!
- Кончики коричневые? → pH или EC, не калий!
