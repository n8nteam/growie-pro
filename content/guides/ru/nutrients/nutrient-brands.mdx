# Обзор популярных линеек удобрений

![Линейки удобрений](/images/guides/nutrient-brands.jpg)

Выбор удобрений — один из самых частых вопросов у начинающих гроверов. На рынке десятки брендов, и каждый обещает лучший результат. Разберём самые популярные и проверенные линейки.

## General Hydroponics (GHE) — Flora Series

Самая популярная линейка в мире. Три компонента, которые покрывают все потребности.

### Состав линейки

- **FloraGro** — для вегетации (N-P-K: 2-1-6)
- **FloraMicro** — база с микроэлементами (N-P-K: 5-0-1)
- **FloraBloom** — для цветения (N-P-K: 0-5-4)

### Плюсы
- Проверена десятилетиями использования
- Гибкая — можно подстраивать пропорции
- Подходит для любого субстрата и гидропоники
- Доступная цена

### Минусы
- Нужно смешивать три компонента
- Не стабилизирует pH автоматически
- Базовая — может потребоваться CalMag и добавки

### Примерная схема

| Стадия | Micro | Grow | Bloom | мл/л |
|--------|-------|------|-------|------|
| Рассада | 0.5 | 0.5 | 0.5 | — |
| Ранняя вега | 1.5 | 2 | 0.5 | — |
| Поздняя вега | 2 | 2.5 | 1 | — |
| Переход | 2 | 1 | 2 | — |
| Цветение | 2 | 0.5 | 3 | — |
| Пик цветения | 1.5 | 0 | 3.5 | — |
| Промывка | 0 | 0 | 0 | — |

## Advanced Nutrients — pH Perfect

Линейка с автоматической стабилизацией pH. Идеальна для тех, кто не хочет возиться с pH-метром.

### Состав линейки

- **Micro** — микроэлементы и азот
- **Grow** — для вегетативной стадии
- **Bloom** — для цветения

### Плюсы
- pH Perfect — автоматически держит pH 5.5-6.3
- Отличный результат «из коробки»
- Много дополнительных продуктов (Big Bud, Overdrive, B-52)
- Хелатные формы микроэлементов

### Минусы
- Дороже конкурентов
- pH Perfect работает только с их линейкой
- Агрессивный маркетинг — не всё нужно покупать
- Много «дополнений», в которых легко запутаться

### Рекомендации
- Базовых трёх компонентов достаточно для отличного результата
- Из добавок полезны: Big Bud (цветение) и B-52 (витамины)
- Не покупай всё — маркетинг не равен необходимости

## Canna

Голландский бренд с продуктами для каждого типа субстрата.

### Линейки по субстрату

- **Canna Terra** — для земли (Vega + Flores)
- **Canna Coco** — для кокоса (A + B)
- **Canna Aqua** — для гидропоники (Vega + Flores)
- **Canna Hydro** — для проточных систем

### Плюсы
- Чёткое разделение по субстратам
- Высокое качество производства (Голландия)
- Отличные добавки (Cannazym, Rhizotonic, PK 13/14)
- Простая двухкомпонентная система

### Минусы
- Высокая цена
- Нужно покупать отдельную линейку для каждого субстрата
- PK 13/14 легко передозировать

### Лучшие добавки Canna
- **Rhizotonic** — корневой стимулятор (первые недели)
- **Cannazym** — расщепляет мёртвые корни
- **PK 13/14** — бустер цветения (осторожно с дозировкой)
- **Boost** — ускоритель метаболизма

## BioBizz

Полностью органическая линейка. Для тех, кто выбирает натуральный подход.

### Состав линейки

- **Bio-Grow** — для вегетации (из сахарной свёклы)
- **Bio-Bloom** — для цветения
- **Top-Max** — стимулятор цветения (гуминовые кислоты)
- **Root-Juice** — корневой стимулятор
- **Fish-Mix** — рыбная эмульсия (вега)

### Плюсы
- 100% органика — сертификация
- Улучшает микрофлору почвы
- Отличный вкус и аромат урожая
- Трудно перекормить

### Минусы
- Только для земли и кокоса (не для чистого гидро)
- Медленный эффект
- Запах (особенно Fish-Mix)
- Может привлекать мошек
- Сложнее контролировать EC

### Советы по BioBizz
- Используй их субстрат Light-Mix или All-Mix
- All-Mix уже содержит удобрения на 2-4 недели
- Начинай кормить минимальными дозами
- pH корректируй до 6.2-6.5

## Plagron

Ещё один голландский бренд с хорошей репутацией.

### Линейки

- **Terra** — минеральные для земли
- **Hydro** — для гидропоники (A + B)
- **Alga** — органические (Grow + Bloom)
- **Cocos** — для кокоса (A + B)

### Плюсы
- Качественное голландское производство
- Alga — хорошая органика по адекватной цене
- Green Sensation — один из лучших PK-бустеров
- Sugar Royal — улучшает вкус

### Минусы
- Менее известен в некоторых регионах
- Меньше информации и отзывов
- Ограниченный выбор добавок по сравнению с AN

## Советы по выбору

1. **Первый гров** → GHE Flora Series или Advanced Nutrients (pH Perfect)
2. **Органика в земле** → BioBizz или Plagron Alga
3. **Гидропоника** → Canna Aqua или GHE Flora
4. **Кокос** → Canna Coco или GHE + CalMag
5. **Бюджет ограничен** → GHE Flora Series

## Общие правила для любого бренда

- **Начни с 50%** рекомендуемой дозы
- **Не смешивай бренды** в базовых удобрениях
- **Добавки от другого бренда** — допустимо (CalMag, стимуляторы)
- **Всегда добавляй в воду**, а не наоборот
- **Порядок добавления**: CalMag → Micro → Grow → Bloom → Добавки → pH
