# График полива и кормления

![Полив и кормление](/images/guides/watering.jpg)

Правильный полив и кормление — основа здорового растения. В этом гайде разберём, когда и сколько поливать, как готовить раствор и на что обращать внимание.

## Основы полива

### Когда поливать?

Самый простой способ — **метод пальца**:
1. Воткни палец в субстрат на 2-3 см
2. Если сухо — пора поливать
3. Если влажно — подожди

### Сколько поливать?

- Полей так, чтобы 10-20% воды вытекло в дренаж
- Это вымывает накопившиеся соли
- Не оставляй растение стоять в воде

### Цикл «влажно-сухо»

Растениям полезно чередование:
- **Влажно**: после полива корни получают питание
- **Сухо**: корни ищут воду и разрастаются
- Не поливай каждый день — дай субстрату подсохнуть

## Контроль pH

pH раствора критически важен для усвоения питательных веществ.

| Субстрат | Оптимальный pH |
|----------|---------------|
| Земля | 6.0 - 6.5 |
| Кокос | 5.5 - 6.5 |
| Гидропоника | 5.5 - 6.0 |

**Как измерять:**
1. Приготовь раствор с удобрениями
2. Измерь pH метром
3. Скорректируй pH Up или pH Down
4. Полей растение
5. Измерь pH дренажа (должен быть в диапазоне)

## Контроль EC (электропроводность)

EC показывает концентрацию солей в растворе.

| Стадия | EC (мСм/см) |
|--------|-------------|
| Рассада | 0.4 - 0.8 |
| Ранняя вега | 0.8 - 1.2 |
| Поздняя вега | 1.2 - 1.6 |
| Раннее цветение | 1.4 - 1.8 |
| Пик цветения | 1.6 - 2.2 |
| Дозревание | 0.0 - 0.4 |

## График кормления по стадиям

### Рассада (неделя 1-2)
- Только чистая вода с правильным pH
- Можно добавить корневой стимулятор (1/4 дозы)

### Вегетация (неделя 3-5)
- Базовое удобрение (Grow): начни с 1/4 дозы
- Постепенно увеличивай до полной дозы
- CalMag при необходимости (особенно на кокосе)

### Цветение (неделя 6-10)
- Перейди на удобрения для цветения (Bloom)
- Добавь PK-бустер с середины цветения
- Следи за пожелтением нижних листьев (нормально)

### Промывка (последние 1-2 недели)
- Только чистая вода
- pH в диапазоне
- Вымывает остатки солей для чистого вкуса

## Частые ошибки

1. **Перелив** — корни гниют без кислорода
2. **Перекорм** — кончики листьев «горят»
3. **Игнорирование pH** — блокировка питательных веществ
4. **Холодная вода** — шок для корней (поливай водой 20-22°C)
