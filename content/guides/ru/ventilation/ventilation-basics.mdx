# Основы вентиляции в гроубоксе

![Система вентиляции гроубокса](/images/guides/ventilation-basics.jpg)

Вентиляция — одна из трёх главных систем гроубокса наряду с освещением и поливом. Без правильного воздухообмена растения страдают от перегрева, недостатка CO2, высокой влажности и застойного воздуха.

## Зачем нужна вентиляция

### Основные функции

**Подача свежего CO2**
- Растения поглощают CO2 для фотосинтеза
- В закрытом пространстве CO2 быстро заканчивается
- Воздухообмен восполняет запас углекислого газа

**Контроль температуры**
- Лампы выделяют много тепла
- Идеальная температура: 22-28°C (вега), 20-26°C (цвет)
- Без вытяжки температура может превысить 35°C

**Регулировка влажности**
- Растения испаряют много воды
- Высокая влажность (>70%) = плесень и гниль
- Приток сухого воздуха снижает влажность

**Укрепление растений**
- Движение воздуха имитирует ветер
- Стебли становятся толще и прочнее
- Улучшается транспорт питательных веществ

## Принцип работы системы

Эффективная вентиляция работает по принципу **отрицательного давления**:

```
Приток (пассивный) < Вытяжка (активная)
```

Это значит, что вытяжной вентилятор должен удалять воздух *быстрее*, чем он поступает через приточные отверстия. В результате:
- Стенки палатки слегка втягиваются внутрь
- Воздух попадает в бокс только через контролируемые отверстия
- Запах не просачивается наружу

## Базовые компоненты

### Минимальная система

Для начинающего гровера нужны всего 3 компонента:

1. **Вытяжной канальный вентилятор**
   - Устанавливается вверху палатки
   - Удаляет горячий воздух и запахи
   - Мощность зависит от объёма бокса

2. **Приточные отверстия**
   - Обычно внизу палатки (встроены в тент)
   - Площадь в 2-4 раза больше вытяжки
   - Можно прикрыть для регулировки

3. **Вентилятор обдува**
   - Обычный осциллирующий вентилятор
   - Создаёт циркуляцию внутри бокса
   - Направлять *НЕ прямо* на растения

### Продвинутая система

Когда бюджет позволяет, добавляются:

- **Угольный фильтр** — устраняет запахи (обязателен при цветении)
- **Приточный вентилятор** — активная подача свежего воздуха
- **Термо/гигро-контроллер** — автоматическое управление климатом
- **Шумоглушители** — снижают шум вентиляции

## Расчёт мощности вентилятора

Базовая формула для расчёта производительности (CFM):

```
CFM = Объём палатки × Кратность воздухообмена
```

**Объём палатки**:
- Размер 80×80×180 см = 0.8 × 0.8 × 1.8 = 1.15 м³ = 40 кубических футов

**Кратность воздухообмена**:
- Без фильтра: 60-80 раз в час (1-1.3 раза в минуту)
- С угольным фильтром: 80-100 раз в час (добавь 30% запаса)

**Пример**:
- Палатка 40 ft³ × 1.5 = **60 CFM** минимум
- С фильтром: 60 × 1.3 = **80 CFM**

## Типичные ошибки новичков

### Слишком слабый вентилятор
**Симптомы**: высокая температура (>30°C), тепловой стресс, растения вялые
**Решение**: выбирай вентилятор с запасом 20-30%

### Нет обдува
**Симптомы**: тонкие слабые стебли, растения падают, плесень на листьях
**Решение**: добавь 1-2 небольших вентилятора внутрь

### Приток больше вытяжки
**Симптомы**: стенки палатки выпирают наружу, запах просачивается
**Решение**: приоткрой приточные отверстия, уменьши приток

### Вентилятор работает 24/7 на максимуме
**Симптомы**: низкая влажность (<30%), растения пересыхают, шум
**Решение**: используй регулятор скорости или таймер

## Схема подключения

Стандартная последовательность для вытяжки:

```
Фильтр → Вентилятор → Воздуховод → Выход
```

Или (более тихий вариант):

```
Фильтр → Воздуховод → Вентилятор → Выход
```

Вентилятор *всегда* тянет воздух через фильтр, а не толкает.

## График работы вентиляции

| Этап цикла | Температура | Влажность | Режим вытяжки |
|------------|-------------|-----------|---------------|
| Рассада | 22-25°C | 65-70% | Постоянно 40-60% |
| Вега | 22-28°C | 50-70% | Постоянно 60-80% |
| Цветение | 20-26°C | 40-50% | Постоянно 80-100% |
