# Выбор вытяжного вентилятора для гроубокса

![Канальный вентилятор в гроубоксе](/images/guides/exhaust-fans.jpg)

Вытяжной вентилятор — сердце системы вентиляции. От правильного выбора зависит температура, влажность, воздухообмен и контроль запаха в гроубоксе.

## Типы вентиляторов

### Канальные (inline) вентиляторы

**Самый популярный тип для гроубоксов**

**Плюсы**:
- Встраиваются в воздуховоды
- Компактные, удобно монтировать
- Тихие при правильной установке
- Легко подключить фильтр

**Минусы**:
- Цена выше осевых
- Требуют регулятор скорости для снижения шума

**Лучшие бренды**:
- AC Infinity (CloudLine серия) — рекомендуем
- Vents (ТТ, ТТ PRO)
- Soler & Palau TD-Silent
- Systemair RVK

### Центробежные (radial) вентиляторы

**Для профессиональных и больших установок**

**Плюсы**:
- Очень мощные (высокое статическое давление)
- Лучше работают с длинными воздуховодами и фильтрами
- Надёжные, промышленное качество

**Минусы**:
- Дорогие ($200-500+)
- Громоздкие
- Шумные без глушителей

**Когда использовать**:
- Палатки больше 2×2 метра
- Длинные воздуховоды (>3gt;3 метров)
- Несколько фильтров в системе

### Осевые вентиляторы

**Не рекомендуются для вытяжки**

**Почему**:
- Низкое статическое давление
- Не могут протолкнуть воздух через фильтр
- Очень шумные

**Где использовать**:
- Только для внутреннего обдува
- Для притока (без фильтра)

## Расчёт производительности (CFM)

### Формула расчёта

```
CFM = (Длина × Ширина × Высота) × Кратность обмена / 60
```

**Пример для палатки 100×100×200 см**:
- Объём: 1 × 1 × 2 = 2 м³ = 70 кубических футов
- Кратность: 60 обменов в час (стандарт)
- CFM: 70 × 60 / 60 = **70 CFM**

### Поправочные коэффициенты

Добавь запас мощности для:

| Фактор | Коэффициент |
|--------|-------------|
| Угольный фильтр | +25-30% |
| Воздуховод &gt;2 метров | +15% за каждый метр |
| Изгибы воздуховода | +10% за каждый поворот 90° |
| HPS/HID освещение | +20% |
| Жаркий климат (>3gt;30°C) | +30% |

**Итоговый расчёт**:
- Базовый CFM: 70
- С фильтром: 70 × 1.3 = 91
- Воздуховод 3м: 91 × 1.15 = 105
- Два поворота: 105 × 1.2 = **126 CFM**

Вывод: для палатки 100×100×200 с фильтром нужен вентилятор **120-150 CFM**.

**Вентилятор**: 200-350 CFM (6"/150 мм диаметр)
**Модели**:
- AC Infinity CloudLine S6 (402 CFM)

## Характеристики вентиляторов

### Уровень шума

Измеряется в децибелах (dB):

- **&lt;30 dB** — очень тихо, как шёпот
- **30-40 dB** — тихо, как фоновый шум
- **40-50 dB** — заметно, как тихий разговор
- **&gt;50 dB** — громко, мешает спать

**Способы снизить шум**:
- Контроллер скорости (снижение до 60-70%)
- Шумоглушители на входе/выходе
- Гибкие виброразвязки для крепления
- Акустические воздуховоды

### Статическое давление

Важный параметр, если используешь фильтр или длинные воздуховоды:

- **Низкое** (<5lt;50 Pa) — только для коротких систем без фильтра
- **Среднее** (50-150 Pa) — стандарт для большинства гроубоксов
- **Высокое** (>1gt;150 Pa) — для сложных систем с фильтрами

### Энергопотребление

Канальные вентиляторы очень экономичны:

| Мощность | Диаметр | Потребление |
|----------|---------|-------------|
| 100 CFM | 4" | 15-25W |
| 200 CFM | 5" | 25-35W |
| 400 CFM | 6" | 40-60W |
| 740 CFM | 8" | 70-100W |

Работая 24/7, вентилятор 4" потребляет всего **0.4-0.6 кВт/ч в день** (~$2-3 в месяц).

## Контроллеры скорости

### Зачем нужен контроллер

- **Снижение шума** — работа на 60-70% в 2-3 раза тише
- **Точная регулировка климата** — контроль температуры и влажности
- **Экономия энергии** — снижение потребления на 30-50%
- **Продление срока службы** — меньше износ подшипников

### Типы контроллеров

**Ручные регуляторы напряжения**
- Самые дешёвые ($10-20)
- Крутишь ручку — меняется скорость
- Минус: могут гудеть на низких оборотах

**Ступенчатые переключатели**
- Фиксированные режимы (40%, 60%, 80%, 100%)
- Без гудения
- Простые и надёжные

**Автоматические термостаты**
- Датчик температуры управляет скоростью
- Удобно: забыл и не паришься
- AC Infinity контроллеры — лучшие на рынке

**Smart-контроллеры с Wi-Fi**
- Управление через смартфон
- Графики, история, алерты
- AC Infinity CloudLine Pro/T-серия

## Схема подключения

### Базовая схема

```
[Фильтр] ←— [Вентилятор] ←— [Воздуховод] ←— [Выход из палатки]
```

Вентилятор **тянет** воздух через фильтр (не толкает).

### Расширенная схема

```
[Фильтр] → [Глушитель] → [Вентилятор] → [Глушитель] → [Выход]
```

Два глушителя максимально снижают шум.

## Монтаж и установка

### Где разместить

**Внутри палатки** (рекомендуется):
- Экономит место снаружи
- Проще монтаж
- Можно использовать ремни или подвесы

**Снаружи палатки**:
- Освобождает пространство внутри
- Легче обслуживать
- Может быть громче (звук не глушится стенками)

### Крепление

- **Подвесы на ремнях** — лучший вариант (гасят вибрацию)
- **Металлические хомуты** — жёстко, может передавать вибрацию
- **На полу** — допустимо для центробежных

## Частые ошибки

**Покупка слишком слабого вентилятора**
- Результат: высокая температура, плохой воздухообмен
- Решение: всегда бери с запасом 20-30%

**Подключение вентилятора неправильной стороной**
- Результат: воздух идёт в обратную сторону
- Решение: проверь стрелку направления на корпусе

**Жёсткое крепление без виброразвязки**
- Результат: вибрация передаётся на каркас, громкий гул
- Решение: используй ремни или резиновые подвесы
