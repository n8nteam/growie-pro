# Перлит, вермикулит, керамзит и другие добавки

![Добавки к субстрату](/images/guides/perlite-vermiculite.jpg)

Добавки к субстрату — это компоненты, которые улучшают физические свойства основной среды: дренаж, аэрацию, влагоёмкость или структуру. Правильно подобранные добавки могут существенно улучшить условия для корней.

## Перлит

Самая популярная и универсальная добавка. Вспученное вулканическое стекло белого цвета.

### Свойства

- Очень лёгкий (плавает в воде)
- Не впитывает воду — удерживает на поверхности
- Создаёт воздушные карманы в субстрате
- pH-нейтральный (6.5-7.5)
- Химически инертный
- Не разлагается

### Фракции

| Фракция | Размер | Применение |
|---------|--------|-----------|
| Мелкий | 1-3 мм | Рассада, семена |
| Средний | 3-5 мм | Универсальный |
| Крупный | 5-10 мм | Дренажный слой, гидропоника |

### Сколько добавлять

- **В землю**: 20-30% от объёма
- **В кокос**: 20-40% от объёма
- **100% перлит**: гидропонные системы, укоренение клонов

### Советы

- Увлажни перлит перед смешиванием — он сильно пылит
- Используй респиратор при работе с сухим перлитом
- Со временем всплывает наверх — нормально
- Заменяй при повторном использовании субстрата

## Вермикулит

Слоистый минерал (слюда), расширенный нагревом. Золотистые чешуйки.

### Свойства

- Впитывает и удерживает воду (до 500% своего веса)
- Удерживает питательные вещества (высокая CEC)
- Лёгкий
- pH-нейтральный (6.5-7.2)
- Содержит немного магния и калия

### Сколько добавлять

- **В землю**: 10-15% для повышения влагоёмкости
- **Для рассады**: 15-20% — удерживает влагу для нежных корней
- **Для проращивания**: можно 100% — семена любят влагу
- **Не используй &gt;20%** в основном субстрате — слишком влажно

### Перлит vs вермикулит

| Параметр | Перлит | Вермикулит |
|----------|--------|-----------|
| Главная функция | Дренаж, аэрация | Влагоёмкость |
| Впитывает воду | Нет | Да (много) |
| Удерживает питание | Нет | Да |
| Когда добавлять | Нужен дренаж | Нужна влага |
| Можно 100% | Да (гидро) | Нет (гниение) |

**Они дополняют друг друга**, а не заменяют. Можно использовать оба.

## Керамзит (глиняные гранулы)

Обожжённая глина в виде круглых пористых шариков.

### Свойства

- Тяжелее перлита
- Не впитывает воду (минимально)
- Отличный дренаж
- Многоразовый — промой и стерилизуй
- pH-нейтральный после промывки

### Применение

- **Дренажный слой** на дне горшка (2-3 см)
- **100% керамзит** для DWC и систем прилива-отлива
- **Мульча** — верхний слой для защиты от мошек и испарения

### Подготовка

1. Промой от пыли и крошек
2. Замочи в pH 5.5 воде на 24 часа (для новых гранул)
3. Стерилизуй при повторном использовании (перекись или кипячение)

## Диатомит (кизельгур)

Осадочная порода из окаменевших диатомовых водорослей.

### Свойства

- Содержит 80-90% кремния
- Абразивный для насекомых
- Впитывает влагу

### Применение

- **Защита от вредителей** — повреждает экзоскелет мошек, трипсов
- Рассыпь по поверхности субстрата (1-2 мм слой)
- Перемешай 5-10% в субстрат как источник кремния
- Работает только в сухом виде — после полива теряет эффективность

## Древесный уголь (биочар)

Обугленная древесина, обработанная при высокой температуре без кислорода.

### Свойства

- Огромная пористая поверхность
- Адсорбирует токсины и тяжёлые металлы
- Дом для полезных бактерий
- Стабилизирует pH
- Не разлагается сотни лет

### Применение

- Добавляй 5-10% в субстрат
- Измельчи до фракции 3-10 мм
- Предварительно замочи в растворе компостного чая (заселит бактериями)
- Отлично работает в живой почве (super soil)

## Агроперлит vs строительный перлит

Важно знать разницу:

| Параметр | Агроперлит | Строительный |
|----------|-----------|-------------|
| Фракция | 1-5 мм | 0.5-2 мм |
| Обработка | Чистый | Может содержать добавки |
| Цена | Выше | Ниже |
| Для растений | Да | Нежелательно |

Используй только агроперлит — строительный может содержать химические добавки.

## Рекомендуемые комбинации

### Для земли

- Земля 65% + перлит 25% + вермикулит 10%
- Идеально для большинства ситуаций

### Для кокоса

- Кокос 70% + перлит 30%
- Золотой стандарт

### Для рассады

- Торф 50% + перлит 30% + вермикулит 20%
- Лёгкий и влагоёмкий

### Для живой почвы

- Земля 40% + компост 20% + перлит 20% + биочар 10% + вермикулит 10%
- Богатый и структурированный

## Практические советы

1. **Перлит — обязательная добавка** для любого субстрата
2. **Вермикулит — осторожно** с количеством, легко переувлажнить
3. **Керамзит на дне** — простой способ улучшить дренаж
4. **Не заменяй перлит пенопластом** — пенопласт не работает так же
5. **Мокрый перлит** не пылит — увлажняй перед смешиванием
