# Типы субстратов: полный обзор

![Типы субстратов](/images/guides/soil-types.jpg)

Субстрат — это среда, в которой живут корни твоего растения. От выбора субстрата зависит стиль полива, тип удобрений и общий подход к выращиванию. Разберём все основные варианты.

## Земля (почва)

Самый традиционный и «прощающий» субстрат. Лучший выбор для первого грова.

### Плюсы

- Буферит pH — прощает ошибки
- Содержит питательные вещества
- Развивает полезную микрофлору
- Проста в использовании
- Совместима с органическими удобрениями
- Лучший вкус и аромат урожая

### Минусы

- Медленнее рост по сравнению с гидро и кокосом
- Может содержать вредителей и патогены
- Тяжёлая — сложнее перемещать горшки
- Сложнее контролировать EC

### Для кого

- Новички
- Органический подход
- Приоритет — вкус и качество

## Кокосовый субстрат (кокос)

Волокна кокосовой скорлупы. Инертный субстрат с отличными свойствами — золотая середина между землёй и гидропоникой.

### Плюсы

- Быстрый рост (близко к гидро)
- Отличная аэрация корней
- Можно использовать повторно
- Лёгкий и удобный
- Устойчив к патогенам
- Экологичный и возобновляемый

### Минусы

- Инертный — не содержит питания, нужны удобрения с первого дня
- Требует CalMag обязательно
- Быстро сохнет — чаще поливать
- Нужно промывать перед использованием (если не буферизованный)

### Для кого

- Гроверы среднего уровня
- Хотят быстрый рост с удобством земли
- Готовы следить за pH и EC

## Перлит

Вспученное вулканическое стекло. Лёгкие белые гранулы.

### Свойства

- Улучшает дренаж и аэрацию
- Не содержит питательных веществ
- Не впитывает воду (держит на поверхности)
- pH-нейтральный

### Применение

- **Как добавка** к земле или кокосу (20-30%)
- **100% перлит** — для гидропонных систем (капельный полив)
- **Для укоренения** клонов

## Вермикулит

Слоистый минерал, расширяющийся при нагреве.

### Свойства

- Удерживает воду и питательные вещества
- Улучшает ёмкость катионного обмена
- Лёгкий
- pH-нейтральный

### Применение

- **Как добавка** к земле (10-20%) — повышает влагоёмкость
- **Для проращивания** семян
- **Не использовать 100%** — слишком влагоёмкий, корни задохнутся

## Минеральная вата (роквул)

Расплавленный базальт, вытянутый в волокна. Популярна в коммерческом выращивании.

### Плюсы

- Стерильная
- Отличное соотношение воздух/вода
- Точный контроль питания
- Многоразовая (с обработкой)

### Минусы

- Нужна предварительная подготовка (замачивание в pH 5.5)
- Не экологичная — не разлагается
- Раздражает кожу и лёгкие при работе
- Только для опытных гроверов

### Для кого

- Опытные гроверы
- Гидропонные системы
- Коммерческое выращивание

## Керамзит (глиняные гранулы, Hydroton)

Обожжённые глиняные шарики. Стандарт для DWC и капельных систем.

### Плюсы

- Многоразовый (промыть и стерилизовать)
- Отличный дренаж
- Не разлагается
- Стабильный pH

### Минусы

- Не удерживает воду
- Не содержит питания
- Нужна постоянная циркуляция раствора
- Тяжёлый

### Для кого

- DWC, капельные системы, системы прилива-отлива
- Как дренажный слой на дне горшка

## Что выбрать?

### Первый гров

**Земля** — готовый грунт (BioBizz Light-Mix или аналог) + перлит 20-30%. Максимально простой старт.

### Хочу быстрый рост

**Кокос** + перлит (70/30) с минеральными удобрениями и CalMag.

### Гидропоника

**Керамзит** для DWC, **минвата** для капельных систем, **перлит** для систем прилива-отлива.

### Органический подход

**Живая почва** (super soil) — земля с органическими добавками и микроорганизмами.

## Практические советы

1. **Новичкам — земля**, всегда. Она прощает ошибки
2. **Добавляй перлит** в любой субстрат для улучшения аэрации
3. **Кокос — отличный шаг** после земли, когда освоишь основы
4. **Не экономь на субстрате** — дешёвая земля с улицы может содержать патогены
5. **Стерилизуй** субстрат при повторном использовании
